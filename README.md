# HR-analysis
Анализ рекрутинговой воронки и времени прохождения этапов найма в зависимости от способа добавления кандидатов («Откликнулся» / «Добавлен»). Включает визуализацию конверсий по этапам (Скриннинг → Интервью с менеджером → Интервью с заказчиком → Оффер) и сегментацию вакансий по ключевым метрикам. Реализовано на Python с возможностью масштабирования для регулярной отчётности.
